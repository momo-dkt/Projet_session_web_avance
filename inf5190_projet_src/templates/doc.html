REST API for the website Infos Installations

/api

/api/installations/{arrondissement} get

get

Gives all the installations of a district.

get /api/installations/{arrondissement}

Gives all the installations of a district.

  • Response

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Existing installations for a district

  • name: required(string)
